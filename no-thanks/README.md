No Thanks! (2004)
=====================================================

Ambil kartu, bayar untuk mengatakan "No Thanks (Tidak Terima Kasih)" atau ambil untuk mendapatkan skor paling sedikit.

* Pemain: 3 - 7 orang
* Waktu: 20 menit
* Umur: 8 tahun keatas
